## Setup
```bash
  # Mac
  brew install ffmpeg
  # Linux
  apt-get install ffmpeg

  
  # Then...
  git clone git@github.com:joshghent/PeepShowBot.git
  cd PeepShowBot
  npm install

  Open .env file and add your token like `TOKEN={ADD_YOUR_DISCORD_TOKEN}`

  npm start
```
